# プロジェクト概要

miniaudioNet は、C のシングルヘッダー軽量オーディオライブラリである [miniaudio](https://miniaud.io/) を .NET から安全かつ直感的に利用できるようにする極小ラッパーです。ネイティブ側では `ma_engine` や `ma_sound` による再生処理を行い、C# からは `MiniaudioEngine` や `MiniaudioSound` を介して再生、停止、ボリューム制御、シーク、再生状態確認といった操作を簡単に記述できます。

## 特徴

- **シングルヘッダー依存**: miniaudio.h のみをネイティブ側でビルド、ランタイム依存が極小。
- **クロスプラットフォーム**: Windows / macOS / Linux 向けのビルド設定とサンプルを同梱。
- **安全な .NET ラッパー**: `SafeHandle` ベースでネイティブリソースを管理し、例外安全な API を提供。
- **NuGet 配布を想定**: ランタイムごとのネイティブバイナリを同梱したパッケージを簡単に生成可能。
- **サンプルとスクリプト**: すぐに動作確認できるコンソールアプリと、ネイティブ資産の自動ビルドスクリプトを提供。

## ディレクトリ構成

```
miniaudioNet/
├── native/                     # CMake ベースのネイティブブリッジ
├── scripts/                    # ビルド補助スクリプト (PowerShell / Bash)
├── src/Miniaudio.Net/          # .NET 8.0 ラッパー (ライブラリ本体)
├── samples/                    # 動作確認用のコンソールアプリ群
├── docs/                       # ドキュメント一式
└── third_party/miniaudio/      # miniaudio.h
```

## 利用シナリオ

- ゲームやマルチメディアアプリでの手軽なオーディオ再生
- GUI ツールでの効果音・BGM 再生
- コマンドラインやサーバーサイドでの自動再生・バッチ処理
- miniaudio を利用した C# アプリケーションのプロトタイピング
